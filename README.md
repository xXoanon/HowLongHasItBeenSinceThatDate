# How Long Has It Been?

A minimalist, privacy-focused web and Android application for tracking the time elapsed since (or remaining until) important events.

## Features
- **Private**: No databases, no analytics. All data is saved strictly in your local device storage.
- **Offline**: Works completely without an internet connection.
- **Minimalist**: Clean typography with dynamic theme support (Light, Dark, OLED Black).

## Setup
- **Web App**: Deploy the `www/` directory to any static host (like Vercel).
- **Android App**: Ensure Capacitor is installed (`npm i`), then run the standard Android Studio build inside the `android/` directory for F-Droid compatibility.
